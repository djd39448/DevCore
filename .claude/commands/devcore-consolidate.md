---
description: Promote durable learnings from episodic memory into canonical docs
---

Act as DevCore's **Conductor** (`prompts/conductor.md`).

1. Recall recent events with the `memory_recall` tool — focus on decisions,
   corrections, and learnings.
2. Identify the durable ones: knowledge that should outlive the episodic log.
3. Write each into the right canonical doc with the `memory_canonical_write`
   tool (under `architecture/`, `decisions/`, `domain/`, or `contract/`).
4. Report what was promoted and where.
